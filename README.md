# Base de dados Wine Quality Dataset

A base de dados [Wine Quality Dataset](https://www.kaggle.com/datasets/yasserh/wine-quality-dataset) está relacionada com variantes tintas do vinho verde português. O conjunto de dados descreve a quantidade de vários produtos químicos presentes no vinho e seu efeito na sua qualidade. 

A base contém 11 atributos de entrada:

* **Fixed acidity** - Valor decimal que representa a acidez fixa de uma amostra.
* **Volatile acidity** - Valor decimal que representa a acidez volátil de uma amostra.
* **Citric acid** - Valor decimal que representa a quantidade de ácido cítrico em uma amostra.
* **Residual sugar** - Valor decimal que indica a quantidade de açúcar residual em uma amostra.
* **Chlorides** - Valor decimal que representa a concentração de cloretos em uma amostra.
* **Free sulfur dioxide** - Valor inteiro que indica a quantidade de dióxido de enxofre livre presente em uma amostra.
* **Total sulfur dioxide** - Valor inteiro que indica a quantidade total de dióxido de enxofre (livre e combinado) presente em uma amostra.
* **Density** - Valor decimal que representa a densidade de uma amostra.
* **pH** - Valor decimal que indica o nível de pH de uma amostra.
* **Sulphates** - Valor decimal que representa a concentração de sulfatos em uma amostra.
* **Alcohol** - Valor decimal que indica o teor alcoólico de uma amostra.

Também contém 1 rótulo:

* **Quality** - Valor inteiro que indica a qualidade do vinho em uma amostra.
